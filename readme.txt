Hotel Management System
-----------
This project is a hotel management system developed using Java FX. System allows employees to make basic reservation
System also has features such as: view room types and prices, add and manage rooms and employees. System saves itself
to the specific data files after every shutdown.
HOW TO USE?
-----------
1. Run the System
2. Enter username and password
3. In the main screen there will be date pickers, use them to see reservations between selected dates.
4. To edit room type and price click Room Management Button
5. Use Employee Management Button to add employees.
6. Use Customer/Reservation button to make bookings
